# Whole-genome sequencing of a sporadic primary immunodeficiency cohort [Core Bash Scripts] 
 
 **Rafal Urniaz, PhD** [![ORCID](https://img.shields.io/badge/ORCID-0000--0003--0192--2165-A6CE39?logo=orcid&logoColor=FFFFFF)](https://orcid.org/0000-0003-0192-2165)

## Key Features
- **Modular Design:** Scripts for filtering, indel removal, format conversion, and association testing.
- **Cambridge Dawn HPC Optimized:** Designed for SLURM array jobs and GNU Parallel.
- **Reproducible:** Centralized configuration via `config.env`.

## Getting Started
1. Clone this repository.
2. Edit `config.env` to set your cluster-specific paths.
3. Run the workflow using `sbatch` or `parallel`.
